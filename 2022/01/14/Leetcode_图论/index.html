1162. 地图分析

你现在手里有一份大小为 N x N 的 网格 grid，上面的每个 单元格 都用 0 和 1 标记好了。其中 0 代表海洋，1 代表陆地，请你找出一个海洋单元格，这个海洋单元格到离它最近的陆地单元格的距离是最大的。

我们这里说的距离是「曼哈顿距离」（ Manhattan Distance）：(x0, y0) 和 (x1, y1) 这两个单元格之间的距离是 |x0 - x1| + |y0 - y1| 。

如果网格上只有陆地或者海洋，请返回 -1。

示例一：

思考：

  • **单源dfs**从一个起点出发，进行dfs遍历，可以得到一个海洋到陆地的最短距离
  • **多源dfs**需要对所有的海洋都进行一次dfs遍历，所有的复杂度可能是$O(n^4)$
  • 因此需要特殊的方法进行处理。

多源dfs

329. 矩阵中的最长递增路径

给定一个 m x n 整数矩阵 matrix ，找出其中 最长递增路径 的长度。

对于每个单元格，你可以往上，下，左，右四个方向移动。 你 不能 在 对角线 方向上移动或移动到 边界外（即不允许环绕）。

示例

思路

  • 搜索路径，经常想到深度优先搜索
  • 采用dfs得到从一个点开始的最长路径
  • 如果对矩阵中每一个点都进行一次dfs搜索明显是愚蠢的
  • 是不是可以记录一个点出发的最长递增路径长度，这样就可以多次使用了
  • 采用dp[i][j]来记录从点[i,j]出发的最长递增路径长度，可以多次调用
