    887. 鸡蛋掉落

    给你 k 枚相同的鸡蛋，并可以使用一栋从第 1 层到第 n 层共有 n 层楼的建筑。

    已知存在楼层 f ，满足 0 <= f <= n ，任何从 高于 f 的楼层落下的鸡蛋都会碎，从 f 楼层或比它低的楼层落下的鸡蛋都不会破。

    每次操作，你可以取一枚没有碎的鸡蛋并把它从任一楼层 x 扔下（满足 1 <= x <= n）。如果鸡蛋碎了，你就不能再次使用它。如果某枚鸡蛋扔下后没有摔碎，则可以在之后的操作中 重复使用 这枚鸡蛋。

    请你计算并返回要确定 f 确切的值 的 最小操作次数 是多少？

    思路：

    • dp[k][N]代表目前有k个鸡蛋，等待确定的楼层有N层
    • dp[k][N] = min(dp[k][N],max(dp[k][N-x],dp[k-1][x-1])+1) 其中， (1<=x<=N-1)
    • 初始条件
      • k=1单独考虑
      • n=0||n=1单独考虑

    逆向思路：

    • f[t][k]我们可以进行t次操作，而且有k个鸡蛋，我们可以找到的最高的楼层n是多少

    • res的结果是找出最小的满足f[t][k]>=n的t即可

    • 地柜方程： f[t][k] = f[t-1][k] +f[t-1][k-1] +1

    走迷宫——边的权值相同

    给定一个n*m的二维整数数组，用来表示一个迷宫，数组中只包含0或1，其中0表示可以走的路，1表示不可通过的墙壁。

    最初，有一个人位于左上角(1, 1)处，已知该人每次可以向上、下、左、右任意一个方向移动一个位置。

    请问，该人从左上角移动至右下角(n, m)处，至少需要移动多少次。

    数据保证(1, 1)处和(n, m)处的数字为0，且一定至少存在一条通路。

    思路：

    • 用BFS可以求得最大值
    • visit[][]记录是否遍历过，也可以利用来记录达到当前点的路径距离
    • 记录路径可以在队列中存入不同的class，也可以开一个pair<int,int> prev[][]来记录得到当前位置的前驱位置

    走迷宫升级版——边的权值不同

    思路：

    采用BFS来求解最短的操作次数，需要记录的的东西：

    • 当前状态 state
    • 当前修改次数

    单纯不采取技巧会超时

    优化：

    • 原先的state是多维数组，可能存在大量的重复计算，采用map来记录状态避免重复
