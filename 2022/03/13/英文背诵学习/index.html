用英语说一下人工智能，机器学习，深度学习的区别

人工智能就是研究如何使计算机去做过去只有人才能做的智能工作。

机器学习是人工智能领域中的一个子集，也是其核心。机器学习过程与人类学习的过程相似，也需要预先对大量的学习资料进行学习来得到模型。当有新的样例时，通过模型对新的样例进行判断，输出结果，从而实现进行预测的功能。

深度学习是基于人工神经网络发展而成，是机器学习的子集。传统机器学习大部分算法需要人类自己寻找特征，而深度学习可以自动从大量数据中生成特征。

总而言之，在我看来，人工智能包含了机器学习，机器学习包含了深度学习。

Artificial intelligence is to study how to make computers do the intelligent work that only talents could do in the past.

Machine learning is a subset of artificial intelligence. The process of machine learning is similar to that of human learning. It also needs to learn a large number of learning materials in advance to get the model. When there is a new sample, judge the new sample through the model and output the results, so as to realize the function of prediction.

Deep learning is developed based on artificial neural network and is a subset of machine learning. Deep learning is a popular machine learning method. Most traditional machine learning algorithms need human beings to find features, while deep learning can automatically generate features from a large amount of data.

In a word, in my opinion, artificial intelligence includes machine learning, and machine learning includes deep learning.
