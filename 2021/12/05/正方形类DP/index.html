221. 最大正方形(动态规划)

在一个由 '0' 和 '1' 组成的二维矩阵内，找到只包含 '1' 的最大正方形，并返回其面积。

示例一：

思考

  • 似乎可以考虑二进制运算，前后行进行并操作
  • 无法思考出如何解决dp[][]代表是什么东西，似乎不太能直接代表答案，应该想出一个有意义的

思路

  • 用 dp[i][j] 表示以 (i, j) 为右下角，且只包含 1 的正方形的边长最大值

  • 递推关系很难得到，需要记忆，因此单独拿出来一个专题

    • dp[i][j] = min{dp[i-1][j],dp[i][j-1],dp[i-1][j-1]}+1 (num[i][j]==1)
    • dp[i][j] = 0 (num[i][j] = 0)

  • 区别：

    • 和思考的区别在于利用dp[i][j]来固定了右下角的元素，矩阵的右下角元素一定是[i,j]
    • 和最长递增子序列类似
